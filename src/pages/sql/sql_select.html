Простые запросы Select

Все операторы записываться в верхнем регистре (По соглашению)

    Операторы
  • SELECT - выбираем из каких столбцов будете осуществяляться выборка данных (* из всех стаблиц)
  • FROM - из какой таблицы будут выбираться данные
  • DISTINCT - Вывод уникальных строк (Всегда ставиться перед столбцом , дубликаты строк не будут показны)
  • COUNT(*) - Подсчет строк в столбце
  • WHERE - фильтрация по условию
  • Если название строки есть ковычки 'US'A' , то экранируем добавление еще одной ковычки 'US''A'
  • AND - Кобинация двух условий ( выводит данные если все условия в true)
  • OR - Или ( выводит данные если хотябы одно условие true)
  • BETWEEN - Интервал включающий верхнию и нижнею границу (Улучшенный вариан AND)
  • IN & NOT IN (Улучшенный вариан OR)
  • ORDER BY - Сортировка столбца по убыванию или возрастанирю (используться после WHERE) (ASC - по возрастанию , DESC - по убыванию)

Полная выборка

Приводит к падению производительности при выборе больших обьмов данных. Желательно вместо * указываем именна столбцов по которым необходимо произвести выборку.

* Считаеться плохим тоном при написании кода. Используеться при разработке.

Выборка всех данных из таблицы product и вывод в консоль
SELECT *
FROM products

Арифметические операции

Перемножаем данные одной колонки на другую
SELECT product_id, product_name, unit_price* units_in_stock
FROM products

При выборе столбцов указывем через запяную , на последнем не ставим

* арифметическая функция переумножаем

DISTINCT

Вывод уникальных строк в столбце city
SELECT DISTINCT city , country
FROM employees

Если необходимо вывести уникальные и множество столбцов указывем через запятую

DISTINCT - Всегда ставиться перед столбцом

COUNT

Вывод Колличества уникальных строк в столбце country
SELECT COUNT(DISTINCT country)
FROM employees

COUNT

Вывод Колличества уникальных строк в столбце country
SELECT COUNT(DISTINCT country)
FROM employees

WHERE

Поиск с условием вывод всех данных по Countru с условием USA
SELECT company_name, contact_name, phone
FROM customers
WHERE country = 'USA'

AND

Вывод данных по условию units_in_stock > 40 and unit_price >25 (Выводит данные если оба условия в true)
SELECT *
FROM products
WHERE units_in_stock > 40 and unit_price >25

OR

Совмещение условий AND и OR
SELECT *
FROM orders
WHERE shipped_date > '1998-04-30' AND (freight 75 OR freight > 150)

BETWEEN

Улучшенная команда AND вывода диапазона со включением 20 и 40
SELECT *
FROM orders
WHERE freight BETWEEN 20 AND 40

IN & NOT IN

Улучшенная команда OR
SELECT *
FROM customers
WHERE country IN ('Mexico','Canada', 'USA')

ORDER BY

Сортировка стран и городов по возрастанию
SELECT DISTINCT country, city
FROM customers
ORDER BY country, city

Like

Фильтрация данных по шаблону Pattern Matching
SELECT first_name, last_name
FROM employees
WHERE first_name LIKE 'A%'

LIMIT

Вывод 10 записей
SELECT first_name, last_name
FROM employees
LIMIT 10

Идет самым последним

Check on NULL

Вывод записей с NULL
SELECT costumer_id, ship_region
FROM orders
WHERE ship_region IS NULL

GROUP BY

Вывод записей с NULL
SELECT costumer_id, ship_region
FROM orders
WHERE ship_region IS NULL
