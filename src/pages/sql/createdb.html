Создание таблице в базе данных Marka
CREATE TABLE Marka
(marka_id int PRIMARY KEY, name varchar(32) NOT NULL );

marka_id - индетификатор в таблице. обычно используеться приставка id к наименованию таблицы

int(integer) - Тип данных цифровой c целым значением

PRIMARY KEY - Уникальный ключь . данные в этом столбце не должны повторяться.

varchar(32) - тип данных и числовой и текстовой с указанием длины.

NOT NULL - Не должен содержать ноль.
